# Scratch / notes in `docs/notes/`

Eric's exploration, drafts, and thinking-out-loud. **NOT load-bearing.**
Read for context to understand intent — never cite as a source of truth.
Decisions live in `docs/adr/`, conceptual content lives in `docs/model/`;

`Created: YYYY-MM-DD` header. Notes that become load-bearing should be
promoted to `model/` or `adr/`; notes left forever become rot.
